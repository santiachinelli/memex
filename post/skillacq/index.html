Update on The Last Day Of The Year

##Pico CSS So I have become proficient in Pico CSS, it’s really just a framework, so it took all of an afternoon to get under my belt.

Deno

As far as the rest are concerned, I’m about halfway through learning how to use Deno. I’m really loving the simplicity of the implementation, even if I’m not a TypeScript nerd. Ironically enough, I can’t get the binary to play nice in setting up a local web server, for some reason. I think it’s got something to do with my network stack, but I have no idea.

D3.js (really A-Frame)

I’ve swapped this out for learning about A-Frame, actually. It’s been an easier on-ramp with it’s simplified concepts and ‘training wheels’ included approach to scene creation. I made one silly meme diorama to start, and have been playing around with figuring out how to get play|pause toggling to work nicely. Play is simple enough, but pausing is a little more complicated. I’m excited that it’s been so easy to set up a virtual gallery space, and would recommend this tool to anyone interested in working in the Web VR/AR/XR space, regardless of skill level.

I have started reading the docs for D3, since a lot of A-Frame’s documentation leaves something to be desired. Plus, I don’t think the codebase has seen an update since about 18 months ago.
